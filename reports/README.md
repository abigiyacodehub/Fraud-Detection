# Reports

This folder stores small, reproducible Task 1 summaries generated by:

```bash
python scripts/run_task1_pipeline.py
```

Generated CSV files are intentionally ignored by Git because they are derived
from the raw datasets. The script creates:

- `data_quality_report.csv`
- `country_fraud_summary.csv`
- `class_imbalance_train_only.csv`
- `task1_pipeline_summary.md`

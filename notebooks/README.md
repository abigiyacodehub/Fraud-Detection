# Notebooks Directory

This directory contains Jupyter notebooks documenting the research, EDA, modeling, and explainability workflows.

## Contents
* `eda-creditcard.ipynb`: Exploratory Data Analysis of the credit card fraud dataset with simplified transaction features.
* `eda-fraud-data.ipynb`: Exploratory Data Analysis of the E-commerce Fraud dataset (mapped with IP to Country lookup).
* `feature-engineering.ipynb`: Feature engineering step for creating structured input features.
* `modeling.ipynb`: Training, hyperparameter tuning, and comparing models for classification.
* `shap-explainability.ipynb`: Analysis of feature importance and model explainability using SHAP.
